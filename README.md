# Жизнь — подлинник «Une Vie» (Maupassant)

Публичная **docshelf-полка**: французский подлинник романа Ги де Мопассана
«Une Vie» (1883), разбитый по главам. Это отдельный репозиторий с raw-ссылками —
ассистент (или человек) может подтянуть нужную главу, не загружая весь роман
целиком (~455 KB).

## Как пользоваться

1. Точка входа — [`INDEX.md`](INDEX.md): навигационный индекс полки.
2. Полный текст лежит в `docs/une-vie/une-vie-1883.md`; он же разложен по
   секциям в `docs/une-vie/une-vie-1883/` — фронтиспис + 14 глав
   (`003-i.md` … `016-xiv.md`).
3. Пофлавный список с raw-ссылками и размерами —
   [`SUBINDEX`](docs/une-vie/une-vie-1883/SUBINDEX.md). Каждая ссылка ведёт на
   `raw.githubusercontent.com`, так что нужную сцену можно подтянуть по одной
   главе.

Полка совместима со спецификацией openshelf (`shelf.yml`, `mode: single`) и
управляется инструментом [docshelf-mcp](https://github.com/ignatenkofi/docshelf-mcp).

> **`INDEX.md` и `SUBINDEX.md` генерируются автоматически** (`rebuild_index`
> из docshelf-mcp) по содержимому `docs/` и настройкам `.docshelf.json`.
> Руками их не редактируют — правится источник в `docs/`, затем полка
> перегенерируется.

## Связь с основным проектом

Полка — публичная часть проекта «Жизнь» ([`ignatenkofi/UneVie`](https://github.com/ignatenkofi/UneVie),
приватный). Основной проект хранит французский подлинник именно здесь, чтобы
не тащить его в приватную репу и раздавать открыто.

## Права

- **Текст Мопассана** (ум. 1893) — **общественное достояние** (public domain):
  во всём мире срок охраны давно истёк.
- **Вклад-компиляция** этого репозитория (разбиение по главам, Markdown-разметка,
  индексы) посвящён общественному достоянию по **[CC0 1.0 Universal](LICENSE)** —
  используйте как угодно, атрибуция не требуется.

Лицензированные переводы сюда не попадают — только подлинник в общественном
достоянии.
